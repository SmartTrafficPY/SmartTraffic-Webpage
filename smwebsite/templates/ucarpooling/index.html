{% trans 'Project Goals' %}

{% trans 'Primary Goal' %}

{% trans 'Design a non-profit carpooling system with a mobile application aimed at closed communities, decreasing the number of vehicles while commuting.' %}

{% trans 'It would benefit the city by reducing the number of vehicles on the road and everything that this causes, like less waiting time at traffic lights, especially during peak hours.' %}

{% trans 'It would benefit the institution that adopts it, by improving the camaraderie among its members and would help with difficulties of parking spaces' %}

{% trans 'It would benefit people by saving fuel money and generating new friendships and connections' %}

{% trans 'Specific Goals' %}

{% trans 'Study the techniques and strategies that apply to the situation of a carpooling system for users of the same institution' %}

{% trans 'Design the carpooling system taking into account what has been studied' %}

{% trans 'Implement the matcher module between people for the system' %}

{% trans 'Analyze the impact of such system in a closed community. In particular at the Catholic University "Nuestra señora de la Asunción"' %}
